## Derma Scanner — AI Skin Cancer Detection Web App

**EfficientNet-B0** · **~80% Accuracy** · **HuggingFace Deployment**

Built and deployed an end-to-end computer vision pipeline for classification of pigmented skin lesions.

### Key Contributions
- Implemented full preprocessing → inference → reporting pipeline.
- Trained EfficientNet-B0 on HAM10000 with balanced evaluation.
- Generated automated **PDF triage reports** for screening workflows.
- Deployed live demo on HuggingFace Spaces.
